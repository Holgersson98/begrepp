Github Flow

Github flow är ett verktyg som är designat för att göra det enkelt när du jobbar med Github och Git, med Github flow blir det enkelt om man jobbar i tex en grupp.
Med hjälp av github flow blir det enkelt att skicka upp delar av sitt arbete och sedan fortsätta med det lite senare istället för att skicka upp allt på en och samma gång.
Det blir också enkelt att göra ändringar i sin kod då det är bara att följa samma steg hela tiden.
Det fungerar så här:

  • Skapa en ny branch
  • Ändringar och lägga till en commit
  • Pull request
  • Granska
  • skicka upp
  • Merga
(w3schools.com, Git Github Flow)

Flexbox

Flexbox är en metod för att tex få saker att lägga sig i en column eller en rad.(MDN-Mozilla,Flexbox)
För att flexbox ska fungera så måste skapa en container som vi sedan använder oss av i vår css fil.Containern består av barnelementen som kallas för items som sedan kan styras av diverse egenskaper.
Med Flexbox blir det enklare att designa din sida och göra den mer responsiv utan att använda några andra verktyg(w3schools.com,CSS Flexbox)

DOM

DOM reprensenterar din sida så att man kan ändra på dokumentets sida dvs strukturen,innehållet och stilen på sidan.
med hjälp av tex javascript så kan man där igenom manipulera din html kod och skriva om den eller ändra den vid vissa funktioner.
det man gör när man vill manipulera koden genom javascript är att man tex kallar på ett id som man har i sin html kod och sedan kan man använda inner.html = för att ändra vad som ska stå där istället (MDN-Mozilla, introduction to the DOM)

Fin kod

Fin kod är ett begrepp som innebär att koden ska vara enkel att läsa igenom och ha en bra struktur.
Det är viktigt att ha en fin kod om man är flera stycken som jobbar med samma arbete så att det är enkelt att se vad den andra har gjort och vad man kan ändra tex.
Och det är också viktigt för din egna skull så att du vet själv vad du har gjort och kommer ihåg din egna kod. Jag skulle vilja säga att det är lite som ett bliotek där allt är lätt att hitta och allt är i ordning.

Ramverk

Ramverk är tillför att göra dit kodande enklare, man kan säga ett det är som ett skelett som användaren sedan kan bygga vidare på och tilldela vissa funktioner för att göra sin sida mer responsiv tex.

Bootstrap:

Bootstrap är ett ramverktyg som först och främst gör din sida mer responsiv.Den gör så att du kan lägga till färger, olika fonter, iconer och ändra storleken på element.

Angular:

Angular är ett ramverktyg som man använder i javascript och det är det mest populära ramverket i javascript.det änvänd för att binda två vägs data så om du har två inputs som är binda tillsammans och du ändrar något i inputens namn så kommer båda inputsen att ändras.
(Codecademy, what is Angular)
